# Premium Scientific Calculator

A high-performance, visually stunning scientific calculator built with Vanilla HTML, CSS, and JavaScript. Featuring a premium dark/light theme, postfix trigonometric functions, and a responsive layout.

## 🚀 Live Demo
**[Click here to try the Calculator](https://web-app-calculator.vercel.app/)**

## ✨ Features
- **Scientific Logic**: Postfix unary operators (sin, cos, tan, log) matching real scientific calculators.
- **Dual Themes**: Beautifully crafted Dark and Light modes with persistent state.
- **Responsive Design**: Premium aesthetics optimized for all screen sizes.
- **Precision**: Floating-point cleanup and safe expression evaluation.
- **Keyboard Support**: Full accessibility with keyboard shortcuts.

## 🛠️ Built With
- HTML5 (Semantic Structure)
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla, Safe Function Evaluation)

## 📖 How to Use
1. **Numbers**: Type numbers using the grid or your keyboard.
2. **Scientific**: For trig/log, type the number first, then press the function (e.g., `30` -> `sin`).
3. **Themes**: Toggle the sun/moon icon at the top to switch between dark and light modes.
4. **Operations**: Use infix operators (+, -, *, /) and parentheses for complex expressions.
