# Interactive Calculator

A clean, lightweight, browser-based calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- **Core Arithmetic**: Addition, subtraction, multiplication, division with operator precedence
- **Memory Functions**: M+, M−, MR, MC
- **History Panel**: Last 10 calculations, click to restore
- **Dark/Light Mode**: Toggle via top-right button
- **Keyboard Support**: Full number pad, operators, Enter, Backspace, Escape
- **Responsive**: Works from 320px to 1920px

## Quick Start

1. Open `index.html` in any modern browser
2. No build step, no dependencies — just open and use

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Number input |
| `+` `-` `*` `/` | Operators |
| `.` | Decimal point |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` or `Delete` | All Clear |
| `%` | Percentage |
